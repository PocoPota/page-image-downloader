# Page Image Downloader

現在開いているページに含まれる画像を収集し、ZIPファイルとしてダウンロードするChrome拡張機能です。

## 使い方

1. Chromeで `chrome://extensions/` を開きます。
2. 右上の「デベロッパー モード」を有効にします。
3. 「パッケージ化されていない拡張機能を読み込む」から、このディレクトリを選択します。
4. 画像を保存したいページを開き、拡張機能アイコンを押します。
5. 「画像を検索してZIPを作成」を押すとZIPファイルを保存できます。

「すべてPNG形式で保存」を有効にすると、WebPやJPEGなどの画像をPNGに変換してZIPに保存します。

## 対象画像

- `<img>` の `src` / `currentSrc`
- `<img srcset>` と `<source srcset>`
- SVG内の `<image>`
- CSSの `background-image` などの `url(...)`

## 注意

ログインが必要な画像や、サイト側の制限がある画像は取得できない場合があります。取得に失敗したURLはZIP内の `manifest.json` に記録されます。
PNG変換はブラウザで読み込める画像が対象です。アニメーション画像は先頭フレームのみPNG化されます。
